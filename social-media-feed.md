# Challenge #3 — Social Media Feed
## Design a schema for a social platform (think Twitter/X):

-Users can follow other users
-Users can create posts (text, optional image URL)
-Users can like posts
-Users can comment on posts
-Comments can be liked too
-Users can repost another user's post (like a retweet)
-Each user has a follower count and following count — think carefully about how to handle this
-Posts have a view count
